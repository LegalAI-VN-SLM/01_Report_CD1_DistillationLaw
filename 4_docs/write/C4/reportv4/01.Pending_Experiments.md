## ⚠️ Chi Tiết Nhóm Thực Nghiệm Đang Hoãn (Pending)

Trong file cấu hình [experiments_v2.yaml](file:///e:/DoCode/1%20VN-Legal-AI/legal-slm-distillation-pdq/distillation_logit_v2/configs/experiments_v2.yaml) (dòng 76–103), nhóm **Group B (KD from Pretrained)** được thiết kế để kiểm chứng xem việc bỏ qua SFT có giúp mô hình tránh được hiện tượng quên lãng thảm họa hay không:

*   **P1:** Qwen3-0.6B (Base) + LoRA + KD (LR = $5.0 \times 10^{-5}$)
*   **P2:** Qwen3-0.6B (Base) + QLoRA + KD (LR = $5.0 \times 10^{-5}$)
*   **P3:** Qwen3-1.7B (Base) + LoRA + KD (LR = $5.0 \times 10^{-5}$)

**Mục tiêu đối chứng:** So sánh đường cong hội tụ và điểm số cuối cùng giữa việc tiếp tục KD trên checkpoint SFT (nhóm A) vs KD trực tiếp trên base model (nhóm P).
